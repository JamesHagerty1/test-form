### Local Setup / Viewing:
#### 1. [Optional] check if your node and react versions match / are compatible:
```bash
$ npm -v
9.8.1
$ npm view react version
18.2.0
```
#### 2. To download and run the app:
```bash
$ git clone https://github.com/JamesHagerty1/test-form.git
$ cd test-form
$ npm install
$ npm start
```
